# 弹窗内编辑功能测试清单

## 快速测试步骤

### 1. 启动应用
```bash
cd personal-knowledge-website
npm run dev
```
访问: http://localhost:5176/

### 2. 打开问题详情
- [ ] 点击任意问题卡片（例如："如何搭建个人博客"）
- [ ] 确认弹窗正常打开
- [ ] 确认显示问题标题、描述、总结、小问题列表

### 3. 测试编辑问题描述
- [ ] 点击"问题描述"区域的"编辑"按钮
- [ ] 确认切换到编辑模式
- [ ] 确认显示Markdown编辑器和工具栏
- [ ] 在编辑器中输入文本
- [ ] 确认预览区域实时更新
- [ ] 测试工具栏功能：
  - [ ] 加粗（**文本**）
  - [ ] 斜体（*文本*）
  - [ ] 代码（`代码`）
  - [ ] 链接
  - [ ] 列表
- [ ] 点击"保存"按钮
- [ ] 确认切换回查看模式
- [ ] 确认修改已保存

### 4. 测试编辑最终总结
- [ ] 点击"THE END - 最终总结"区域的"编辑"按钮
- [ ] 确认切换到编辑模式
- [ ] 在编辑器中输入文本
- [ ] 确认预览区域实时更新
- [ ] 点击"保存"按钮
- [ ] 确认切换回查看模式

### 5. 测试取消编辑
- [ ] 点击"编辑"按钮进入编辑模式
- [ ] 修改一些内容
- [ ] 点击"取消"按钮
- [ ] 确认弹出确认对话框（如果有修改）
- [ ] 确认取消后内容恢复原状

### 6. 测试自动保存
- [ ] 进入编辑模式
- [ ] 修改内容后等待3秒
- [ ] 确认显示"保存中..."或"已保存"指示器
- [ ] 确认自动保存功能正常

### 7. 测试图片上传（如果实现）
- [ ] 点击工具栏的图片按钮
- [ ] 确认打开图片上传对话框
- [ ] 上传图片或输入图片URL
- [ ] 确认图片Markdown插入到编辑器

### 8. 测试状态切换
- [ ] 在查看模式下，点击状态下拉框
- [ ] 切换问题状态（未解决/解决中/已解决）
- [ ] 确认状态更新

### 9. 测试小问题列表
- [ ] 确认在查看模式下显示小问题列表
- [ ] 确认在编辑模式下隐藏小问题列表
- [ ] 点击"添加小问题"按钮（查看模式）
- [ ] 确认功能正常

### 10. 测试关闭弹窗
- [ ] 点击"返回"按钮
- [ ] 确认弹窗关闭
- [ ] 点击"×"按钮
- [ ] 确认弹窗关闭
