# ⚠️ 重要注意事项

## 🔑 关于 Token 和数据存储

### 最重要的一点

**数据存储在 GitHub Gist 中，不在 Token 中！**

```
Token = 钥匙 🔑
Gist = 房子 🏠
数据 = 家具 📦

钥匙丢了可以配新的，房子和家具还在！
```

### 更换 Token 后数据不会丢失

✅ **完全安全**：
- Token 只是访问凭证
- 数据存储在 Gist 中
- 系统会自动检测你的 Gist
- 自动下载所有数据

### 实际流程

```
旧 Token 过期 → 生成新 Token → 输入应用
                                    ↓
                            系统自动检测 Gist
                                    ↓
                            自动下载数据
                                    ↓
                            一切恢复正常！✨
```

## 💡 快速提示

1. **Token 过期了？** → 重新生成即可，数据不会丢
2. **换设备了？** → 配置同一个账号的 Token，数据自动同步
3. **清除浏览器数据了？** → 重新配置 Token，数据自动恢复
4. **担心数据丢失？** → 定期使用"导出数据"功能备份

## 🛡️ 数据安全

你的数据有三重保护：
1. **GitHub Gist**：云端存储，永久保存
2. **本地缓存**：浏览器 LocalStorage，离线可用
3. **版本历史**：GitHub 保存所有历史版本

**只要 GitHub 账号和 Gist 还在，数据就是安全的！**
