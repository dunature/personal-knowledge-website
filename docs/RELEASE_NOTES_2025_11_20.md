# 发布说明 - 2025-11-20

## 功能改进

### 🔧 Gist ID 验证增强

#### 更灵活的格式支持
- **支持 20-40 位十六进制字符**（之前只支持 32 位）
- 自动去除首尾空格
- 大小写不敏感（自动转换）

#### 智能错误分类
系统现在会将错误分为三大类，并提供针对性的解决方案：

1. **格式错误**
   - 提供正确格式示例
   - 详细的格式要求说明
   - 常见错误提示

2. **网络错误**
   - 网络连接问题诊断
   - 重试建议
   - 故障排除步骤

3. **验证错误**
   - 区分 404（不存在）、403（无权限）、401（认证失败）等
   - 针对每种错误的具体解决方案
   - 详细的错误原因说明

#### 改进的用户界面
- 实时格式验证
- 字符计数显示（例如：32/40）
- 错误类型图标
- 多行错误消息支持
- 可展开的详细信息

### 🔧 访客模式同步功能

#### 新增功能
- **访客模式下可从 Gist 拉取最新数据**
- 设置页面添加同步按钮
- 支持只读同步，保持数据最新
- 同步进度显示

#### 使用方法
1. 进入设置页面
2. 在"Gist 数据管理"区域找到同步按钮
3. 点击"从 Gist 拉取数据"
4. 等待同步完成
5. 页面自动刷新显示最新数据

#### 权限控制
- **访客模式**：只能拉取数据（只读）
- **拥有者模式**：支持双向同步（拉取和推送）
