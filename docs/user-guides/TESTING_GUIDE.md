## 🐛 常见问题排查

### 问题1: 页面空白
**解决方案**:
- 检查浏览器控制台是否有错误
- 确认所有依赖已安装（`npm install`）
- 清除浏览器缓存（Cmd+Shift+R / Ctrl+Shift+R）

### 问题2: 弹窗无法打开
**解决方案**:
- 检查浏览器控制台错误
- 确认lucide-react已安装
- 刷新页面重试

### 问题3: Markdown预览不显示
**解决方案**:
- 确认marked和highlight.js已安装
- 检查控制台错误
- 刷新页面

### 问题4: 自动保存不工作
**解决方案**:
- 等待至少3秒
- 确认修改了内容
- 检查右下角通知区域
