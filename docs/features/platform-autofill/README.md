# 平台自动填充功能

自动从 YouTube、Bilibili、GitHub 等平台获取资源信息（标题、作者、封面等）。

## 🎯 支持的平台

- ✅ YouTube - 获取标题、作者、封面
- ✅ Bilibili - 获取标题、UP主、封面
- ✅ GitHub - 获取仓库名、作者、描述

## 🚀 快速开始

1. 添加资源时输入 URL
2. 选择对应的资源类型
3. 点击"自动填充"按钮
4. 系统自动填充信息
