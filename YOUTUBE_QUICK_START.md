# YouTube 视频信息自动获取 - 快速开始

## 🎯 功能说明

在添加 YouTube 视频资源时，系统可以自动获取：
- ✅ 视频封面（缩略图）
- ✅ 视频标题
- ✅ 频道名称（作者）

无需手动查找和复制这些信息！

## 📝 使用步骤

### 方法 1：自动填充（推荐）

1. 点击主页的"添加资源"按钮
2. 填写以下信息：
   - **链接**：粘贴 YouTube 视频 URL
   - **类型**：选择"YouTube 视频"
3. 点击"自动填充"按钮
4. 系统自动填充：
   - ✅ 视频标题（如果标题为空）
   - ✅ 频道名称（如果作者为空或为"未知"）
   - ✅ 视频封面
5. 检查信息，点击"保存"

### 方法 2：保存时自动获取

1. 点击主页的"添加资源"按钮
2. 填写以下信息：
   - **标题**：输入视频标题
   - **链接**：粘贴 YouTube 视频 URL
   - **类型**：选择"YouTube 视频"
   - **封面**：留空（不填写）
3. 直接点击"保存"
4. 系统自动获取并保存封面

## 🎬 支持的视频平台

- ✅ YouTube（完全支持）
- ⚠️ Bilibili（使用占位图）

## 📋 支持的 URL 格式

YouTube 支持以下所有格式：

```
https://www.youtube.com/watch?v=VIDEO_ID
https://youtu.be/VIDEO_ID
https://www.youtube.com/embed/VIDEO_ID
https://www.youtube.com/watch?v=VIDEO_ID&feature=xxx
```

## 💡 使用技巧

1. **先输入 URL 再选择类型**：这样可以立即看到"自动填充"按钮
2. **智能填充**：系统只会填充空白字段，不会覆盖你已经输入的内容
3. **一键获取**：点击"自动填充"按钮，标题、作者、封面一次性获取
4. **自定义信息**：如果不喜欢自动获取的信息，可以手动修改

## ⚠️ 注意事项

- "自动填充"按钮只在选择视频类型且输入 URL 后显示
- YouTube 视频信息通过 oEmbed API 获取，无需 API 密钥
- 如果网络请求失败，至少会获取视频封面
- Bilibili 视频目前只支持封面获取（未来会支持完整信息）
- 系统不会覆盖已填写的字段，只填充空白内容

## 🔧 故障排除

### 问题：按钮没有显示

**解决方案**：
1. 确认已输入视频 URL
2. 确认已选择"YouTube 视频"或"Bilibili 视频"类型
3. 刷新页面重试

### 问题：信息获取失败

**解决方案**：
1. 检查 URL 是否正确
2. 检查网络连接
3. 如果 API 失败，至少会获取封面图
4. 可以手动输入标题和作者信息

### 问题：作者名称不正确

**解决方案**：
1. YouTube API 返回的是频道名称
2. 如果不满意，可以手动修改作者字段
3. 系统不会覆盖已填写的作者信息
